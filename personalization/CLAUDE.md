# 个性化

泓泓看的主题和外观定制系统。

## 架构

```
personalization/
├── themes/                    # 主题（容器）
│   └── default/              # 默认主题
│       ├── README.md         # 主题概览
│       ├── strokes/          # 描边（主题内）
│       ├── effects/          # 特效（主题内）
│       ├── icons/            # 图标（主题内）
│       └── (配色见 Theme.swift)
└── minigames/                # 小游戏（独立于主题）
    └── M01-dino-runner.md
```

**主题是顶层容器**，一个主题包含：描边 + 特效 + 图标 + 配色。切一个主题，这四个维度全部换掉。

**小游戏独立于主题**，不受主题切换影响（属于登录页面功能）。

## 添加新主题

在 `themes/` 下新建目录，如 `themes/space/`，包含：
- `README.md` 主题说明
- 各维度子目录（strokes/, effects/, icons/）
- 对应代码模型更新（IconSet, GameTheme）
